# Installation

## Maven

```xml
<dependency>
  <groupId>games.mythical</groupId>
  <artifactId>ivi-sdk-java</artifactId>
  <version>2.6.0</version>
</dependency>
```

## Gradle Groovy DSL

```groovy
implementation 'games.mythical:ivi-sdk-java:2.6.0'
```

## Gradle Kotlin DSL

```kotlin
implementation("games.mythical:ivi-sdk-java:2.6.0")
```
